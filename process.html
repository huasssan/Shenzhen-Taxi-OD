Taix OD

Data Extraction and Visualization

Mobility and geographical data have become more accessible due to the widespread adoption of geo-technologies. This project focus on taxi GPS data to understand the taxi movement pattern in Shenzhen.
READ MORE

Introduction

This visualization project is divided into three sections: 1) data processing, 2) data visualization, and 3)web establish.

Data processing

Extract taxi GPS data and filter data which outside Shenzhen bounday. Then extract single trajectory from the dataset as it has openstatus and time column, and divide it into vacant status and carrying status. Finally we can extraxt the origin and destination location, and export it as json file.

Data Visualization

Deck.gl and Mapbox GL JS are frequently used together to create compelling geospatial visualizations.Combining the two JavaScript libraries together can produce seamlessly matched results. In this project, Mapbox GL JS provides basemap for deck.gl's layers. Arclayer, Scatterlayers and Hexagonlayer are used to visualize OD flow and taxi demands. Kepler.gl is used for visualize the taxi trips within one day.

Web Establish

First,I use Adobe Illustator and Photoshop to design webpage. Then export each picture and write code in a html file to frame the webpage stacture. After setting the skeleton, CSS file is used to define the display properties for each HTML element.
